Embed the Unspeakable World

Drop a chrome-less, interactive sky scene into your blog, course site, or Notion page. Single <iframe>, no script tag, no API key. Pan, pinch, scroll-wheel — your readers explore right inside the iframe.

URL pattern

Append ?embed=1 to any unspeakable-world.pages.dev URL. Both query (?embed=1) and hash (#embed) detection work — either one on its own is enough. 

https://unspeakable-world.pages.dev/?embed=1#solar?focus=Saturn
https://unspeakable-world.pages.dev/?embed=1#viewer?ra=10.68&dec=41.27&fov=8
https://unspeakable-world.pages.dev/?embed=1#viewer

Iframe snippet

Paste this into your HTML, Markdown article, or Notion "Embed" block. Set the height to whatever fits your layout — 540px or 60vh are sensible defaults. 

<iframe
  src="https://unspeakable-world.pages.dev/?embed=1#viewer"
  width="100%"
  height="540"
  style="border:0;border-radius:12px"
  loading="lazy"
  allow="fullscreen">
</iframe>

Three example embeds

Sky Atlas mode is the most fully chrome-less surface. Solar Flight and Universe modes still surface their mode-specific controls; the install banner and PWA prompts are suppressed across every mode.

sky atlas Saturn ringed pin ?embed=1#viewer?ra=309&dec=-15&fov=10
sky atlas Andromeda (M31) ?embed=1#viewer?ra=10.68&dec=41.27&fov=8
sky atlas ISS live tracker ?embed=1#viewer

Responsive sizing

The viewer is fluid — it fills 100% of the iframe at any aspect ratio. For mobile, prefer an aspect-ratio container with a sensible min-height (around 320px) so the canvas still has room to render. Avoid fixed pixel widths. 

<div style="aspect-ratio:16/9;min-height:320px">
  <iframe
    src="https://unspeakable-world.pages.dev/?embed=1#viewer"
    style="width:100%;height:100%;border:0;border-radius:12px"
    loading="lazy"
    allow="fullscreen">
  </iframe>
</div>

What's in / out

Included in embed mode

  • The 3D scene — drag, pinch, scroll-wheel still work
  • Background sky tiles, stars, planets, and the live ISS marker
  • A tiny "Unspeakable World ↗" link in the bottom-right that opens the full app in a new tab

Hidden in embed mode

  • Top bar (mode picker, share, snapshot, search, panels)
  • Bottom HUD (FOV chips, wavelength bar, time strip)
  • SIMBAD inspector / Wikipedia popovers
  • About, shortcuts, tutorial, and the PWA install banner
