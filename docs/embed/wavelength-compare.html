Same sky, three wavelengths — drop-in embed

Three iframes locked to the same RA/Dec/FOV but rendering different surveys: optical (DSS2 colour), infrared (2MASS), and X-ray (Chandra deep field). Perfect for "see how a galaxy looks different in different light" blog posts.

The default coordinates point at the Galactic Centre. Edit ra/dec/fov to any other target — values are J2000.0 decimal degrees.
